Create the file:- touch test.txt
Edit the file:- vi test.txt
to put the in Staging:- git add test.txt
to commit the file with a  message (ex: what is modified or committed):- git commit -m test.txt
to update or upload the file to main Repo in git hub:- git push

create the folder and push from local to GitHub repo commands as follows
create the folder:- mkdir sample
once the folder  is created in the local system initiate git: git init
then to put the folder in Staging:-  git add
add a message to commit: git commit -m "message"
to push the folder which is created locally use this command:- git add remote origin https://github.com/seenu38396/sample1.git
git push origin main
